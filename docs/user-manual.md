# 桐叶售电代理系统操作手册

> 适用版本：重构版 v1.0  
> 适用对象：系统管理员、财务人员、业务员、代理商

---

## 1. 系统简介

桐叶售电代理系统是面向售电公司及其代理商的一站式业务管理平台。系统覆盖 **代理管理、客户管理、电价管理、用电数据、询价报价、利润管理、分润结算及定时任务** 等核心模块，实现业务流程数字化、结算自动化、数据可视化。

### 1.1 核心模块一览

| 模块 | 核心功能 | 对应前端路由 |
| --- | --- | --- |
| 代理管理 | 代理列表、代理树、分润配置、代理费查询 | `/agents` |
| 客户管理 | 客户信息、账户状态、价差变更、合同管理 | `/customers/*` |
| 电价管理 | 基础电价、电网电价、批发电价、市场分配价、其他费用 | `/prices/*` |
| 用电数据 | 日用电、小时数据、96 点数据、数据转换、曲线模板、导入任务 | `/consumption/*` |
| 询价报价 | 询价创建、报价、状态流转、导出 | `/inquiries/*` |
| 利润管理 | 日/月利润、统计汇总 | `/profits/*` |
| 分润结算 | 分润配置、代理费审批、结算付款 | `/commissions/*` |
| 定时任务 | 日利润计算、月汇总、分润结算、电价生效、合同到期提醒 | 系统后台 |

## 2. 登录与首页

### 2.1 登录步骤

1. 打开浏览器（推荐 Chrome / Edge），输入系统生产地址。
2. 在登录页输入 **用户名** 和 **密码**。
3. 点击【登录】，系统校验 JWT 身份及 RLS 行级权限。
4. 登录成功后跳转至 `/dashboard` 首页。

### 2.2 首页说明

- **关键指标卡片**：今日用电量、本月利润、待审批代理费、即将到期合同数。
- **快捷入口**：新增客户、创建询价、查看利润、导入用电数据。
- **消息中心**：定时任务执行结果、合同到期预警、审批待办。
- **菜单导航**：左侧为模块菜单，顶部为账号与退出。

## 3. 代理管理

### 3.1 查看代理

- **菜单路径**：代理管理 > 代理列表（`/agents`）
- **列表字段**：代理名称、上级代理、等级、状态、联系人、联系电话、创建时间。
- **常用操作**：
  - 查看详情：点击代理名称进入详情页。
  - 编辑：修改代理基础信息、联系人、账户状态。
  - 启用/禁用：控制代理能否登录系统。
  - 重置密码：为代理账号重置初始密码。
- **代理树视图**：切换“树形视图”可直观展示上下级关系，支持展开/收起。

### 3.2 配置分润

- **菜单路径**：代理管理 > 分润配置
- 分润规则可按 **全局默认 + 代理单独** 两级配置；若代理未单独配置，则使用全局默认值。

#### 配置步骤

1. 进入分润配置页面。
2. 选择需要配置的代理商。
3. 选择结算周期（月 / 季度）。
4. 选择分润类型并设置参数：
   - **固定比例**：直接填写分润百分比。
   - **阶梯比例**：按电量或利润区间设置不同比例。
   - **保底/封顶**：设置最低保障金额或最高上限金额。
5. 设置生效日期与失效日期（可选）。
6. 点击【保存】，规则即时生效。

#### 分润类型说明

| 分润类型 | 适用场景 | 说明 |
| --- | --- | --- |
| 固定比例 | 标准代理商 | 按固定百分比从利润或电费中计提代理费 |
| 阶梯比例 | 大客户代理 | 电量/利润越高，适用比例越高或越低 |
| 保底 + 封顶 | 风险共担模式 | 设置最低代理费保障及最高代理费上限 |

### 3.3 查询代理费

- **菜单路径**：代理管理 > 代理费
- **筛选条件**：代理商、结算周期、结算状态。
- **列表字段**：代理商、结算周期、应结金额、已结金额、结算状态、操作。
- **操作**：查看结算明细、导出对账单、标记付款。

## 4. 客户管理

### 4.1 新增客户

- **菜单路径**：客户管理 > 客户列表（`/customers/list`）

#### 操作步骤

1. 点击【新增客户】。
2. 填写客户基础信息（带 `*` 为必填）：
   - 客户名称（企业全称）
   - 统一社会信用代码
   - 所属行业、电压等级
   - 联系人、联系电话
   - 用电地址
3. 选择所属代理商（可选，用于后续分润结算）。
4. 设置默认电价方案。
5. 点击【保存】，客户状态默认为“正常”。

#### 关键字段说明

| 字段 | 说明 |
| --- | --- |
| 客户名称 | 营业执照上的企业全称 |
| 统一社会信用代码 | 用于合同、开票及与原系统对账 |
| 电压等级 | 决定可选择的电价方案 |
| 所属代理 | 关联代理商，影响分润归属 |
| 账户状态 | 正常 / 停用 / 待签约 |

### 4.2 变更价差

- **菜单路径**：客户管理 > 价差变更（`/customers/price-change`）

#### 操作步骤

1. 选择目标客户。
2. 点击【新增价差变更】。
3. 填写变更信息：
   - 生效日期
   - 变更后价差（元/kWh，可正可负）
   - 变更原因
4. 上传审批单或说明附件（可选）。
5. 提交审批。
6. 审批通过后，系统将在 **生效日期** 自动应用新价差；历史数据不受影响。

### 4.3 管理合同

- **菜单路径**：客户管理 > 合同管理（`/customers/contract`）

#### 操作步骤

1. 进入合同列表，点击【新增合同】。
2. 选择客户与合同类型（售电合同 / 代理合同）。
3. 填写合同关键信息：
   - 合同编号
   - 合同期限（开始日期 / 结束日期）
   - 约定电量
   - 电价条款（可关联已维护的电价方案）
4. 上传合同扫描件（PDF / Word / 图片）。
5. 点击【保存】，系统自动生成合同到期提醒。

> **到期提醒**：合同到期前 30 天、15 天、7 天，系统会通过消息中心提醒相关人员。

## 5. 电价管理

菜单路径：电价管理 > 基础电价 / 电网电价 / 批发电价 / 市场分配价 / 其他费用。

### 5.1 基础电价

- **路由**：`/prices/base`
- 维护售电公司与客户结算的基准价格，支持 **分时电价**（尖峰、高峰、平段、低谷）。

#### 操作步骤

1. 进入基础电价页面，点击【新增】。
2. 选择电压等级与用电分类。
3. 填写各时段电价（元/kWh）。
4. 设置生效日期。
5. 保存。

### 5.2 电网电价

- **路由**：`/prices/grid`
- 维护电网代购电价或输配电价，通常按月度/季度更新。
- 支持手动录入或导入电网发布的电价文件。

### 5.3 批发电价

- **路由**：`/prices/wholesale`
- 维护电力批发市场采购价格。
- 可按日/月维护，支持 96 点价格曲线导入。

### 5.4 市场分配价

- **路由**：`/prices/allocation`
- 维护市场分配清算价格，用于利润计算中的市场侧收入或成本。

### 5.5 其他费用

- **路由**：`/prices/other`
- 维护政府性基金、附加费、偏差考核费、输配电损等。
- 支持按固定值或公式计算。

#### 电价维护通用流程

| 步骤 | 操作说明 |
| --- | --- |
| 1 | 进入对应价格表页面 |
| 2 | 点击【新增】或【导入】 |
| 3 | 填写或上传价格数据 |
| 4 | 设置生效日期 |
| 5 | 保存并提交 |
| 6 | 定时任务 `price-effective` 在生效日自动启用最新价格 |

> **注意**：如需修改已生效价格，请新增一条未来生效日期的记录，原记录将在新记录生效日自动失效，系统始终按 **生效日期取最新** 的价格参与计算。

## 6. 用电数据

菜单路径：用电数据 > 日用电 / 小时数据 / 96 点数据 / 数据转换 / 曲线模板 / 导入任务。

### 6.1 录入日用电

- **路由**：`/consumption/daily`
- 用于录入客户每日 24 小时电量数据。

#### 操作步骤

1. 选择客户与日期。
2. 逐小时填写电量，或点击“复制上一日”快速填充。
3. 系统自动汇总当日总电量。
4. 点击【保存】。

> 支持按模板批量导入，详细参见 6.5 导入任务。

### 6.2 导入 96 点数据

- **路由**：`/consumption/point96`
- 96 点数据即每 15 分钟一个点的负荷曲线，一天共 96 个点。

#### 操作步骤

1. 在“96 点数据”页面下载系统提供的导入模板（Excel）。
2. 按模板填写：客户编码、日期、00:00 ~ 23:45 各时点电量。
3. 进入 **导入任务**（`/consumption/import-task`），点击【上传文件】。
4. 选择数据类型为“96 点数据”。
5. 选择日期范围与覆盖策略（跳过 / 覆盖 / 追加）。
6. 提交任务。
7. 在导入任务列表查看处理进度、成功数、失败数及错误日志。

### 6.3 使用曲线模板

- **路由**：`/consumption/template`
- 系统预置典型负荷曲线模板，如工业、商业、居民等。

#### 操作步骤

1. 选择模板类型（如“工业三班制”）。
2. 输入客户当日总电量。
3. 系统自动按模板比例拆分为 24 小时或 96 点曲线。
4. 预览拆分结果，确认无误后点击【保存为实际数据】。

### 6.4 数据转换

- **路由**：`/consumption/conversion`
- 用于在日用电、小时数据、96 点数据之间互相折算。

#### 常见场景

| 场景 | 源数据 | 目标数据 | 转换规则 |
| --- | --- | --- | --- |
| 仅有日总电量 | 日用电 | 24 小时 | 按模板比例拆分 |
| 汇总小时数据 | 小时数据 | 日用电 | 各小时相加 |
| 生成小时曲线 | 96 点数据 | 小时数据 | 每 4 个点相加 |
| 拆分日总电量 | 日用电 | 96 点 | 按模板比例拆分 |

#### 操作步骤

1. 选择源数据类型与目标数据类型。
2. 选择客户与日期范围。
3. 选择转换规则（平均拆分 / 按模板拆分 / 等比例）。
4. 点击【执行转换】。
5. 查看转换结果预览，确认后点击【保存】。

### 6.5 导入任务

- **路由**：`/consumption/import-task`
- 统一管理所有用电数据导入任务，包括 96 点、日用电、小时数据等。
- 支持查看任务状态、下载错误报告、重新执行失败任务。

## 7. 询价报价

### 7.1 创建询价

- **路由**：`/inquiries/create`

#### 操作步骤

1. 选择客户：可直接选择系统已有客户，或临时录入潜在客户信息。
2. 填写用电信息：
   - 预计年用电量
   - 电压等级
   - 用电时段（峰/平/谷占比，可选）
   - 期望合同期限
3. 上传历史电费单或用电曲线（可选，帮助报价）。
4. 选择需要报价的电价方案。
5. 填写询价有效期。
6. 点击【提交询价】，询价状态变为“待报价”。

#### 询价状态流转

| 状态 | 说明 | 可执行操作 |
| --- | --- | --- |
| 待报价 | 业务员尚未报价 | 报价、编辑、作废 |
| 已报价 | 已生成报价单 | 编辑报价、发送客户、确认合作 |
| 客户确认 | 客户已同意报价 | 生成合同、转入客户 |
| 已合作 | 合同已签订 | 查看详情、归档 |
| 已失效 | 超过有效期未确认 | 重新询价 |

### 7.2 提交报价

- **路由**：`/inquiries` 或 `/inquiries/:id`

#### 操作步骤

1. 在询价列表中找到状态为“待报价”的记录，点击【报价】。
2. 选择电价方案：
   - 基础电价
   - 价差
   - 其他费用（政府性基金、偏差考核等）
3. 系统根据用电数据自动试算电费与利润。
4. 填写报价有效期、付款方式、备注。
5. 生成报价单，点击【保存并发送】。
6. 客户收到报价单后可在系统中确认或驳回。

### 7.3 确认合作

1. 客户确认报价后，业务员在询价详情页点击【确认合作】。
2. 系统自动生成客户档案草稿。
3. 补充合同信息并保存后，客户正式转入“客户管理”模块。

### 7.4 导出

- **支持导出内容**：
  - 询价记录列表（Excel）
  - 单个报价单（PDF）
  - 批量报价单（ZIP）
- **操作**：勾选记录 > 点击【导出】> 选择格式 > 下载。

## 8. 利润管理

### 8.1 查看日/月利润

- **路由**：`/profits/monthly`、`/profits/summary`

#### 数据生成机制

| 类型 | 生成时间 | 依赖数据 |
| --- | --- | --- |
| 日利润 | 每日凌晨由定时任务自动计算 | 用电数据、电价、合同 |
| 月利润 | 次月 1 日凌晨自动汇总 | 日利润 |
| 统计汇总 | 实时查询 | 日/月利润 |

#### 列表字段

- 客户名称
- 结算日期 / 结算月份
- 用电量（kWh）
- 销售收入（元）
- 采购成本（元）
- 其他费用（元）
- 毛利润（元）
- 净利润（元）

#### 利润计算逻辑

| 项目 | 计算公式 |
| --- | --- |
| 销售收入 | 用电量 ×（基础电价 + 价差 + 其他费用） |
| 采购成本 | 用电量 × 批发电价 / 市场分配价 |
| 毛利润 | 销售收入 - 采购成本 |
| 净利润 | 毛利润 - 代理费 - 偏差考核 - 其他成本 |

### 8.2 统计

- **按客户汇总**：查看每个客户的累计电量、收入、利润。
- **按代理汇总**：查看每个代理商带来的客户利润。
- **按月趋势**：折线图展示利润变化，支持同比、环比。
- **导出**：支持导出利润明细与汇总报表（Excel）。

## 9. 分润结算

### 9.1 分润配置

- **路由**：`/commissions/config`
- 配置不同维度（代理商、客户、产品）的分润规则。
- 优先级：**客户单独规则 > 代理单独规则 > 全局默认规则**。

#### 配置步骤

1. 选择配置维度（全局 / 代理商 / 客户）。
2. 选择分润类型（固定比例 / 阶梯 / 保底封顶）。
3. 设置结算周期与生效时间。
4. 保存。

### 9.2 代理费审批

- **路由**：`/commissions/approval`

#### 操作步骤

1. 系统每月定时任务 `commission-settlement` 自动生成代理费结算单。
2. 财务人员/管理员进入审批列表。
3. 核对结算单：代理商、结算月份、客户清单、应结金额。
4. 点击【通过】或【驳回】。
5. 若驳回，需填写原因；业务员可在代理费列表修改后重新提交。

### 9.3 结算付款

- **路由**：`/commissions/fees`

#### 操作步骤

1. 审批通过的结算单进入待付款列表。
2. 财务导出付款清单，线下完成付款。
3. 回到系统，找到对应结算单，点击【标记已付款】。
4. 上传付款凭证（银行回单截图等）。
5. 结算单状态变为“已结清”。

## 10. 常见问题

### Q1：登录后页面空白或报错？

- 检查浏览器是否启用了 JavaScript。
- 清除浏览器缓存后重新登录。
- 确认账号权限包含对应的 RLS 访问策略。
- 检查网络是否可访问前端构建的静态资源。

### Q2：导入 96 点数据失败？

- 核对模板格式：必须包含客户编码、日期、00:00 ~ 23:45 的 96 列电量。
- 确保无空行、无合并单元格。
- 检查客户编码是否在系统中存在。
- 查看导入任务的“错误日志”定位具体失败原因。

### Q3：利润数据为空或计算错误？

- 确认已维护对应日期的电价表（基础电价、批发电价等）。
- 确认已录入或导入对应客户的用电数据。
- 确认客户已签订有效合同。
- 检查定时任务 `daily-profit-calc` 是否正常运行。

### Q4：代理费金额与预期不符？

- 检查分润配置是否已生效，且生效时间覆盖结算月份。
- 检查客户所属代理关系是否正确。
- 检查合同期限是否覆盖结算月份。
- 检查是否启用了保底/封顶规则。

### Q5：合同到期未收到提醒？

- 确认定时任务 `contract-expiry-reminder` 已启用。
- 检查消息中心通知设置是否开启。
- 检查合同结束日期是否填写正确。

### Q6：如何修改已经生效的电价？

- 不要直接修改历史记录，应新增一条未来生效日期的价格记录。
- 系统在生效日自动切换为新价格，历史数据不受影响。

### Q7：询价报价单如何发送给客户？

- 在询价详情页点击【保存并发送】，系统会生成报价单链接或 PDF。
- 可将 PDF 下载后通过邮件/微信发送给客户，也可引导客户登录系统查看。

### Q8：代理商只能看到自己的数据吗？

- 是的。系统通过 RLS 行级安全策略控制数据可见范围：
  - 代理商：仅能看到自己的客户、代理费、利润分成。
  - 业务员：能看到自己负责的客户与询价。
  - 管理员/财务：可查看全部数据。
